hexo的安装及使用

下载node.js

安装博客框架

(1) 安装cnpm npm install -g cnpm –registry=https://registry.npm.taobao.org

(2) cnpm install -g hexo -cli

生成博客

(1) 创建博客的文件夹

(2) 在文件夹中打开cmd,输入hexo init

hexo生成

(1) hexo n “标题”(生成的博客.md文件在source中)

(2)清理:hexo clean

(3)生成:hexo g

(2) 开启博客:hexo s
